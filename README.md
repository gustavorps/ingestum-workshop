# Ingestum Workshop

Translations: [pt-BR]

## Requirements
- Linux (OS ou VM)
- Docker
- Conda ou Mamba
- VSCode (recommended) because of "Developing inside a Container" feature


## Getting Started

1. Clone this repo
2. Clone ingestum to `src` dir and install
    ```
    $ git clone https://gitlab.com/sorcero/community/ingestum src/ingestum
    $ pip install -e src/ingestum
    ```
